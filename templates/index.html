Steps :

Using Web scraping to collect data from https://www.cardekho.com/

Data Integration:

Data is gather for each car Model and Data varies in size and type.Here Two operations such as joining and appending are performed to combine information from different data sets. Joining tables allows you to combine the information of one observation found in one table with the information that you find in another table. Appending or stacking tables is effectively adding observations from one table to another table.

Data Cleaning:
1. Correct Data entry based on our requirements
2. Remove Duplicate rows and columns to avoid Data Leakage
3. Fill the Nan values by using Statistical value based methods beacuse missing values follows the MCAR type
Transforming data:
1. Feature Selection
2. Feature Scaling
3. Feature Encoding

Exploratory Data Analysis refers to the process of performing analysis on data so as

1. To Discover patterns
2. To Spot anomalies,
3. To test hypothesis and to check assumptions (this is optional : based on the algorithm we are using we can use this steps)

Machine Learning models are created.I try many models like DecisionTree,RandomForest,KNN,GradientBoosting

Once the model are build, the performance of the each model is evaluated and pick the best one.In these case i got GradientBoosting as the best algorithm

After testing the model and improving the efficiency, it is deployed on all users.
